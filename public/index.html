Welcome to the Deployed App!

A containerized Node.js application deployed automatically via GitHub Actions.

This web application is the core deliverable representing the target of our automated CI/CD pipeline. Every commit pushed to the main branch is built, tested, packaged into a Docker image, pushed to Docker Hub, and deployed seamlessly to hosting.

Core App Features Installed:

  • Production-grade Health Checks: Available at /health returning structured diagnostics.
  • Live Commit Integrity: Integrated build argument exposing the Git SHA at /api/version.
  • Real-time Pipeline Dashboard: Visual monitoring of GitHub Actions at /dashboard.
  • Containerized Isolation: Packaged via multi-stage alpine Docker containers.
